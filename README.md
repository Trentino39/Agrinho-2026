# 🌱 Agro Forte — Futuro Sustentável

Projeto desenvolvido para o **Agrinho 2026** com o tema:

> **“Agro forte, futuro sustentável: equilíbrio entre produção e meio ambiente.”**

O site apresenta soluções sustentáveis para a agricultura moderna, incentivando a preservação ambiental através da tecnologia, conscientização e inovação.

---

# 📚 Sobre o Projeto

O objetivo deste projeto é demonstrar como a agricultura pode crescer de maneira sustentável utilizando:

- Tecnologia no campo
- Economia de água
- Energia limpa
- Reflorestamento
- Preservação ambiental
- Agricultura inteligente

O site foi desenvolvido utilizando:

- ✅ HTML5
- ✅ CSS3
- ✅ JavaScript
